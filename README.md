🛡️ Classificador de Nível de Herói

O objetivo é aplicar conceitos fundamentais como variáveis, operadores e estruturas de decisão em JavaScript.

📋 Objetivo

O sistema deve receber o nome e a quantidade de experiência (XP) de um herói e, com base em uma estrutura de decisão, determinar o seu nível seguindo a tabela abaixo:

🏆 Tabela de Classificação

XP	Nível
Até 1000	Ferro
1001 – 2000	Bronze
2001 – 5000	Prata
5001 – 7000	Ouro
7001 – 8000	Platina
8001 – 9000	Ascendente
9001 – 10000	Imortal
Acima de 10000	Radiante

🛠 JavaScript

Node.js (para execução via terminal)

📈 Possíveis Melhorias Futuras

- Permitir entrada de dados pelo usuário
- Classificar múltiplos heróis
- Criar sistema de ganho de XP
- Transformar em aplicação web simples
